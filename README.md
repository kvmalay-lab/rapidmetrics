# 🚀 RapidMetrics: Strategic Sales Dashboard

> **Impact:** Reduced manual reporting time from **4 hours to 15 minutes**.

### 📊 Live Demo
👉 **[View Live Dashboard](https://rapidmetrics.lovable.app)**

### 🏆 The Challenge
Sales leadership was spending **4 hours every week** manually compiling reports from disparate CSV files, leading to delayed insights and reactive decision-making.

### 💡 The Solution
I built **RapidMetrics**, an interactive, real-time dashboard using **React** and **Recharts**. 
- **Dynamic Filtering:** Instantly slice data by Region, Product, and Time.
- **Automated Insights:** The system automatically highlights anomalies (e.g., "15% drop in West Region") and suggests actions.
- **One-Click Reporting:** Generates professional PDF summaries instantly.

### 🛠 Tech Stack
- **Frontend:** React, Tailwind CSS, Framer Motion
- **Visualization:** Recharts
- **Utilities:** `jspdf` (PDF Export), Lucide React (Icons)
- **Data:** Synthetic dataset simulating 5 years of sales performance

### 📸 Key Features
- **Mobile-First Design:** Fully responsive layout optimized for "10-second recruiter scans."
- **Data Lineage:** Transparent documentation on data logic and scalability.
- **Performance:** Optimized state management for instant filter updates.
